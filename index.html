Free Wall

Create dynamic grid layouts without gaps.

How it works

Base on the width (or height) of container and the width (height) of cell unit. It was create a virtual matrix. Scans matrix at each cell it will find free cell around to made a free area, then try to fits a block element into. In case no block can fit the gap, it will resize a block to fill the gap (that is one of options).

Getting started

Freewall require jQuery framework, for some nice animations effect, it require browser support CSS3.

Options

  • animate
  • block
  • cell
  • fillGap
  • gutter
  • selector

All options is parameters of Reset method

animate : boolean

True to make block move with animation

block.flex : boolean

True to made block can resize a bit to fit

False of block has class 'fixed' to keep size of block can't change

cell.width : number

The width of unit, base on it will build grid container.

cell.height : number

The height of unit, base on it will build grid container.

fillGap : boolean

True will made layout without gap.

gutter : mixed

The spacing between the blocks. Default is number, but can set it with 'auto' value.

selector : string

Get all blocks to made layout

Example 
							    var wall = new freewall('.free-wall');
							    wall.reset({
							      selector: '.brick',
							      animate: true,
							      block: {
							        flex: true
							      },
							      cell: {
							        width: 150,
							        height: 150
							      },
							      fillGap: false
							    });

Events

  • onResize
  • onSetBlock

onResize

Registry callback when browser resize.

Example 
							    var wall = new freewall('.free-wall');
							    wall.reset({
							      selector: '.brick',
							      onResize: function() {
							        this.fitWidth();
							      }
							    });

onSetBlock

Registry callback when arrange block.

Methods

  • fitHeight
  • fitWidth
  • fitZone
  • fixSize
  • reset
  • setFilter
  • unsetFilter

fitHeight(height)

Made the layout fit with limit height

fitWidth(width)

Made the layout fit with limit width

fitZone(width, height)

Made the layout fit with limit width and height

fixSize(object)

Set a block with special width or height

Example 
							    var wall = new freewall('.free-wall');
							    var dna = $(".free-wall .flex-layout");
							    wall.unsetFilter();
							    wall.fixSize({
							        block: dna,
							        width: 150,
							        height: 150
								});
							    wall.fitWidth();

reset(options)

Config options for render layout.

Example 
							    var ewall = new freewall(demo);
							    var demo = $(".example");
							    ewall.reset({
							      selector: '.cell',
							      animate: false,
							      block: {
							        flex: 1
							      },
							      cell: {
							        width: 150,
							        height: 150
							      },
							      fillGap: true
							    });

setFilter(selector)

Fillter blocks to show

unsetFilter()

Made all block to show
